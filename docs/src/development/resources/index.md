# GIS Resources

Where to find internal resource collections (projects, templates, data, assets etc)

External resource links:

| Resource                                |                                                                                                      About                                                                                                      |
| --------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| [Google Cloud Platform](./gcp/index.md) | Google Cloud consists of a set of physical assets, such as computers and hard disk drives, and virtual resources, such as virtual machines (VMs), that are contained in Google's data centers around the globe. |
| [GeoServer](https://geoserver.org/)     |                                        Open-source server for sharing geospatial data via OGC standards (WMS, WFS, WCS, WMTS); integrates well with PostGIS and GeoTIFF.                                        |
| [GeoNode](https://geonode.org/)         |                         Open-source platform to publish, catalog, and manage geospatial data/maps with metadata, permissions, and a web UI; commonly paired with GeoServer and PostGIS.                         |
| [QGIS Docs](https://docs.qgis.org/)     |                                                                                 Official QGIS user and developer documentation.                                                                                 |
| [GDAL/OGR](https://gdal.org/)           |                                                                    Geospatial data abstraction library for raster/vector I/O and processing.                                                                    |
| [MapLibre GL JS](https://maplibre.org/) |                                                                     Open-source web mapping library for vector tiles and interactive maps.                                                                      |
